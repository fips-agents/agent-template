# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly. Do not open a public issue.

Contact [@rdwj](https://github.com/rdwj) on GitHub.

You should receive a response within 72 hours. If the vulnerability is confirmed, a fix will be prioritized and a security advisory published.

## Scope

This project provides agent templates. Security concerns may include:

- Credential exposure in scaffolded projects
- Insecure defaults in configuration templates
- Vulnerabilities in BaseAgent's tool dispatch or MCP client integration
- Container image security (base image, dependency versions)
